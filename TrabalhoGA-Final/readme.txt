Para executar o trabalho basta executar o arquivo .jar.

A pasta TeoInfoTGA é a pasta que contém o projeto java. Todo o código encontra-se na pasta.

Exemplo de arquivos para serem compactados foram disponibilizados:
- Huffman Adaptativo: texto_para_huffman.txt
- Golomb: texto_para_golomb.txt

Ao executar o arquivo .jar, uma tela será exibida, possibilitando a seleção do algorítmo a ser utilizado para codificação.
A partir do momento que o algorítmo é selecionado, existe um botão 'How to Use' que contém todas as informações relevantes para a execução.

Obs: Devido ao tamanho dos arquivos à serem compactados, pode ocorrer lentidão.
